配置项参数

属性 类型 缺省值 描述
type string null 通知的类型。可取值：success，warning，error。对应成功通知、警告通知和错误通知。
show boolean true 是否显示通知
disappear number 3 通知显示几秒后自动消失，仅在成功通知中有效。默认显示3秒钟后自动消失

样式

样式 描述
px-alert-on-top 通知出现在顶部导航下面，内容页面顶部。
