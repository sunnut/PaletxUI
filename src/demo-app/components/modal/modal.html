配置项参数

属性 类型 缺省值 描述
title string Title 弹出框标题
submit function 无 弹框默认按钮，确认按钮触发的事件
setStyles object 无 弹框自定义样式拓展
showDefaultFooter boolean true 是否显示默认按钮
type string 无 警告框类型：exclamation
message string 确认要删除选中项吗？ 警告框提示信息（上），设定type后才会生效
messageDown string 删除选中项可能导致服务不可用。 警告框提示信息（下），设定type后才会生效
warnConfirm string 确认要删除选中项吗？ 警告弹出框底部确定按钮文字，默认‘删除’，设定type后才会生效
warnCancel string 删除选中项可能导致服务不可用。 警告弹出框底部取消按钮文字，默认‘取消’，设定type后才会生效
