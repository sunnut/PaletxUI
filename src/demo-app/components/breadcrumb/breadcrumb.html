配置项参数

属性 类型 缺省值 描述
links 对象数组 空数组 link对象包含四个属性配置：name-显示的名称，url-跳转的路由，relative-是否为相对路由， params-路由参数。 当relative为true时，url使用相对路由。默认使用绝对路由。
name string 空 显示的名称
url string 空 跳转的路由
params 对象 空 键值对,附带参数
relative boolean false 当relative为true时，url使用相对路由。默认使用绝对路由。
