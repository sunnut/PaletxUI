Text Input 文本框

单行文本框

{{inputCode}}

配置项参数

属性 类型 缺省值 描述
id string null id
name string null 名称
ngModel 双向绑定变量
placeholder string null 占位文本
hintLabel string null 提示性描述
required boolean false 是否为必填项。添加required属性时，为必填项，可以进行表单必填校验。
disabled boolean false 是否禁用输入框的数据输入。添加disabled属性时，禁止数据输入，为只读模式。
hintLabel string null 提示性描述，当输入框激活时显示。
notShowOption boolean false 是否显示（可选）提示。添加notShowOption属性时，不显示（可选）提示，不进行表单必填校验。
shortInput boolean false 是否采用短输入框样式。添加shortInput属性时，提供短输入框样式宽度为120px，默认输入框宽度为400px。
width string 400px 自定义输入框宽度, width包括前缀和单位的宽度。
prefix string null 是否显示前缀。如prefix="https://"。
prefixWidth string 70px 自定义前缀框宽度。短输入框下前缀框宽度为40px，默认输入框下前缀框宽度为70px。
unit string null 是否显示单位，仅在数值输入即type="number"场景下有效。如unit="GB"。
unitWidth string 70px 自定义单位框宽度，短输入框下单位框宽度为40px，默认输入框下单位框宽度为70px。
precision number 0 设定小数点后最多保留位数，仅在数值输入即type="number"场景下有效。
step number 1 设定步长，仅在数值输入即type="number"场景下有效。
max number|string null 最大值校验，仅在数值输入即type="number"场景下有效。如max="100"或max=100。
min number|string null 最小值校验，仅在数值输入即type="number"场景下有效。如min="100"或min=100。
maxLength number|string null 最大字符串长度校验。如maxLength="100"或maxLength=100。
minLength number|string null 最小字符串长度校验。如minLength="100"或minLength=100。
ipv4 string null IPV4格式校验。添加ipv4属性时，可以进行表单IPV4格式校验。
